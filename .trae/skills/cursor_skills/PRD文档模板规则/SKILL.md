# PRD文档模板规则

## 模板结构

```markdown
# 产品需求文档（PRD）

## 1. 产品概述
### 1.1 背景
### 1.2 目标
### 1.3 范围
### 1.4 术语表

## 2. 用户分析
### 2.1 目标用户
### 2.2 用户角色
### 2.3 用户画像

## 3. 功能需求
### 3.1 功能模块划分
### 3.2 功能详细描述

## 4. 非功能需求
### 4.1 性能要求
### 4.2 安全要求
### 4.3 兼容性要求

## 5. 业务流程
### 5.1 业务流程图
### 5.2 流程说明

## 6. 接口需求
### 6.1 接口列表
### 6.2 接口详情

## 7. 数据字典
```

## 命名规范
- 文件名：`{项目名}_PRD_{版本}.docx`
- 版本格式：`1.0.0`、`1.1.0`、`2.0.0`
