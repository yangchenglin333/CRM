# 智能任务调度

## 任务类型

| 类型 | 说明 | 优先级 |
|------|------|--------|
| 紧急重要 | 需要立即处理 | P0 |
| 重要不紧急 | 需要计划处理 | P1 |
| 紧急不重要 | 可以委托处理 | P2 |
| 不重要不紧急 | 可以忽略 | P3 |

## 任务拆分原则

1. **单一职责**: 每个任务只做一件事
2. **可估算**: 任务能在规定时间内完成
3. **可测试**: 任务有明确的完成标准
4. **独立性强**: 任务之间依赖最小化

## 任务状态

| 状态 | 说明 |
|------|------|
| 待开始 | 任务已创建，尚未开始 |
| 进行中 | 任务正在执行 |
| 阻塞中 | 任务等待外部条件 |
| 已完成 | 任务达成目标 |
| 已取消 | 任务被终止 |

## 任务跟踪

使用 TodoWrite 工具跟踪任务进度：
```json
{
  "id": "1",
  "content": "任务描述",
  "priority": "high",
  "status": "in_progress"
}
```

## 任务优先级

- **high**: 必须完成，影响主线
- **medium**: 建议完成，影响体验
- **low**: 可选完成，不影响功能
